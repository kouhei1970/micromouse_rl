# Phase 1: Open Space Control (基礎制御)

## 目的 (Purpose)
障害物のないオープンスペースにおいて、マイクロマウスの基礎的な運動制御（直進、旋回、目標地点への移動）を学習・確立すること。
ここで学習した「Low-Level Controller」は、以降のPhase（迷路探索など）において、上位エージェントからの指令（「直進せよ」「右折せよ」など）を実行するアクチュエータとして機能する。

## 環境仕様 (Environment Specifications)
*   **空間:** 障害物（壁・柱）のない広大な平面（Open Space）。
*   **床 (Floor):**
    *   テクスチャ: 市松模様 (Checkerboard)。
    *   グリッドサイズ: 0.18m (180mm) 四方。これはPhase 2の1区画サイズと視覚的に統一されている。
    *   物理特性: 平坦な摩擦のある床。
*   **マウス:**
    *   2輪独立駆動型マイクロマウスモデル。
    *   センサー: オドメトリ（位置・角度）、IMUなどを想定。

## タスク (Tasks)
1.  **直進安定化:** 指定された距離を真っ直ぐ進む。
2.  **旋回制御:** 指定された角度だけ正確に回転する。
3.  **Waypoints追従:** ランダムな目標地点へ移動する。

## 成果物 (Deliverables)
*   **モデルファイル:** `models/phase1_open.zip` (PPOなどで学習済みのLow-Level Controller)
*   **検証動画:** `outputs/phase1_open/` に生成される動画。

## 注意事項 (Notes)
*   **壁・柱の禁止:** このPhaseでは、壁や柱などの障害物を設置してはならない。あくまで「何もない空間」での制御精度を高めることが目的である。
*   **視覚的統一:** 床の模様などのビジュアルは、将来的なVisionベースの学習を見越してPhase 2以降と統一するが、物理的な障害物は配置しない。
